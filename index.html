Restaurant App


In this software engineering project, I and five other group members were tasked with creating a set of apps to serve a fictional restaurant business. The project consists of three UI applications, one for the customers, one for the kitchen, and one for the waiters, as well as a server application that allows communication between all UI apps.

Written using: Java, Swing

UNT Degree Planning App


This is my senior software engineering capstone project. For this project, I and three other group members worked with a professor at UNT to create an app for her. Our app is designed to allow students to plan out their UNT degree and view the information about the classes for that degree.

Written using: Django, Python, Postgresql, HTML, JavaScript, CSS
